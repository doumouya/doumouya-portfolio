# rbac-explorer

![ci](https://github.com/doumouya/rbac-explorer/actions/workflows/ci.yml/badge.svg)

An interactive, fully **client-side** visualization of scoped-ownership access control. Pick an actor
and watch the nodes they can reach light up across the org tree; click any node to grant or revoke
their membership and see reach recompute live.

The rule it makes tangible: **a membership grants the node *and everything scoped beneath it*, never
the parent.** That rule is a pure Rust resolver compiled to WebAssembly — the same rule the companion
[build-engine](https://github.com/doumouya/build-engine-demo) enforces server-side with a recursive
SQL CTE. Everything runs in the browser; nothing leaves the page.

## Status
- **`rbac-core`** — the pure, cycle-safe reach resolver. 5 tests, compiles to `wasm32`. ✅
- **`rbac-wasm`** — the `WasmGraph` binding (`new(ids, parents)` + `reachable(seed)`). Verified through wasm. ✅
- **`index.html`** — a working **single self-contained file** (62 KB). Open it → pick an actor → explore
  reach, grant/revoke by clicking. Verified end-to-end. ✅ *(Compact reference UI; the polished version
  — a real graph layout + org editing — is the next step via Claude Design from [`docs/fe-brief.md`](docs/fe-brief.md).)*

## Layout
```
rbac-explorer/
  index.html            # the built single-file demo (open it; generated by tools/build.sh)
  crates/core/          # rbac-core — the pure reach resolver
  crates/wasm/          # rbac-wasm — the wasm-bindgen binding
  web/                  # the reference UI source (style.css, body.html, app.js)
  tools/build.sh        # compiles wasm + inlines everything into index.html
  docs/spec.md          # the access model + the wasm JS API contract
  docs/fe-brief.md      # the front-end brief (for Claude Design)
```

## Build / try
```sh
cargo test -p rbac-core                      # the resolver's behaviour
sh tools/build.sh                            # (re)build the single-file index.html  (--dev to skip wasm-opt)
# then just open index.html in any browser — no server, fully offline
```
